# ram-chan 👋

### 「AIに育てられた男として将来インタビューされる予定」🤖✨

ram-chanのページへようこそ！
私はAIとガッツリ思考を同期させて、「バイブコーディング」で爆速でサービスを生み出すエンジニア兼クリエイターです。デザインもやります！

## 🧠 Development Philosophy

**「AIは自分の脳の鏡。AIの調子が悪い時は、自分の調子が悪い時」**

これが私の開発モットーです。適当なプロンプトを投げるのではなく、GeminiやChatGPTとじっくり壁打ちをして、システム構想やアーキテクチャをしっかり練り上げます。その設計図をCursorに叩き込んで、最速でプロトタイプを作り上げるのが私のスタイルです！

## 🛤️ Developer Journey & Design Skills

コードを書くだけじゃなく、デザインにも強いこだわりを持っています。

- **確かな基礎知識:** 2008年に国家資格である**基本情報技術者試験**に合格。独学ベースではありますが、コンピュータサイエンスの基礎はしっかり押さえています。
- **C言語からのスタート:** 28〜30歳の時にC言語を独学して、プログラミングの基礎と論理的思考を叩き込みました。
- **デザインへのこだわり:** 30歳以降、アメブロのCSSカスタマイズをきっかけにWebフロント（HTML/CSS）の世界へ。Photoshopも独学して、一時は**フォトマニピュレーターを目指した**ほどです！なので、UI/UXや見た目のデザインにも自信があります。
- **そして現在:** JavaScriptはちょっとだけ、PHPに至っては手書きだと `echo` しか書けません（笑）。でも、過去に培った基礎知識とデザインセンス、そして最強の相棒「AI（LLM）」を掛け合わせることで、言語や技術の壁を越えてフルスタックな開発を実現しています！
